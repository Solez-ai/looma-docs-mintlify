Looma is a fully in-browser video and image editor designed for creating professional product demos and mockups. Built with Next.js, Three.js, and FFmpeg.wasm, it runs entirely in the browser — no server, no backend, no uploads. Everything from screen capture to MP4 encoding happens locally on the user's machine.

## Get up and running

<Steps>
  <Step title="Clone and install">
    Clone the Looma repository and install its dependencies with npm.

    ```bash
    git clone https://github.com/Solez-ai/looma.git
    cd looma
    npm install
    ```
  </Step>
  <Step title="Start the dev server">
    Launch the Next.js development server.

    ```bash
    npm run dev
    ```
  </Step>
  <Step title="Open the editor">
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser. The editor loads instantly — no login or account required.
  </Step>
</Steps>

<Note>
  Looma is entirely client-side. There is no server, no database, and no network calls during editing. All media processing — including screen recording, video encoding, and image export — runs directly in your browser using WebAssembly and browser-native APIs.
</Note>
